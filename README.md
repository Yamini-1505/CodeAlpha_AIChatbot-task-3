# AI Chatbot (Java)

This is a simple rule-based chatbot built using Java.

## Features
- Responds to greetings
- Answers basic predefined questions
- Provides simple information
- Exit command to stop chatbot

## Supported Inputs
- hello / hi
- your name
- java
- help
- exit

## Concepts Used
- Conditional statements (if-else)
- Loops (while)
- String handling
- Scanner class

## Output
Interactive console chatbot conversation.

## Purpose
To simulate basic AI-like behavior using rule-based logic.
